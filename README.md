# Secure Cloud Authentication and Data Protection System

A secure cloud-based web application developed using Flask that protects user data against SQL injection attacks and data leaks using multi-layer security mechanisms.

---

## Security Features

- SQL Injection Prevention (Parameterized Queries)
- Password Hashing using bcrypt
- AES-256 Encryption (Fernet)
- Secure Session Management
- Protected User Authentication System
